Perfil de egreso

La formacion que ofrece la carrera de Técnico en Programación permite al egresado, a través de la articulación de seberes de diversos campos.realizar actividades dirigidas a la :


  • Instalación y desarollo de software de aplicación utilizando programacion estructurada y orientada a objetos en ambientes web y móviles, con almacenamiento permanente de datos,
  • Así como la configuración y administración de plataformas de e-learning y
  • Comercio electronico.

    • Desarrolla software de aplicacion utilizando programacion orientada a objetos,con almacenamiento persistente de los datos
    • Desarrola aplicaciones web y moviles
